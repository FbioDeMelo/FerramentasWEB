Sobre o Sistema

Este sistema de conversão de arquivos foi desenvolvido para oferecer uma solução completa, gratuita e acessível para transformar documentos entre formatos como PDF, DOCX, imagens (PNG/JPG) e muito mais. É ideal para estudantes, profissionais e empresas que buscam agilidade e praticidade, sem depender de plataformas pagas ou conexão constante com a internet.

Com uma interface clara e objetiva, o projeto visa automatizar tarefas comuns como conversões, unificações e extrações de arquivos, entregando resultados com alta qualidade e rapidez diretamente no navegador.

Tecnologias Utilizadas

  • HTML5 e CSS3 – Estrutura e estilo responsivo da interface
  • JavaScript – Funcionalidades dinâmicas e validações no lado do cliente
  • Python (Flask) – Framework web leve para o back-end e roteamento
  • pdf2docx – Conversão de arquivos PDF para DOCX
  • LibreOffice (via subprocess) – Conversão de arquivos DOCX para PDF
  • PyMuPDF (fitz) – Extração de páginas de PDF como imagens
  • Pillow (PIL) – Manipulação e conversão de imagens para PDF
  • pypdf – Mesclagem de múltiplos arquivos PDF
  • zipfile – Compactação de múltiplos arquivos convertidos
  • Werkzeug – Manipulação segura de nomes de arquivos
